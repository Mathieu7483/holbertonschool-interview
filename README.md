## Description

This repository contains various algorithms and data structure problems designed to help students prepare for technical interviews. These projects are part of the curriculum at **Holberton School** to train for the technical challenges encountered during job hunts.

Each project focuses on a specific type of problem and is implemented in Python, adhering to strict coding standards (PEP 8) and execution requirements.

## General Requirements

All projects in this repository adhere to the following strict requirements:

### Interpreters & Compilers
* All files are interpreted/compiled on **Ubuntu 14.04 LTS**.
* All Python files are written and tested using **`python3` (version 3.4.3)**.

### File Structure
* All files must end with a new line.
* The first line of all Python files must be exactly `#!/usr/bin/python3`.
* A `README.md` file is **mandatory** at the root of each project folder.

### Style & Documentation
* Code uses the **PEP 8 style** (specifically version `1.7.x`).
* All code must be documented.
* All files must be **executable** (`chmod u+x filename`).
